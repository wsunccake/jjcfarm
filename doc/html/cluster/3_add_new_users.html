加入新的cluster使用者

1.ip位址
2.指令
3.若mail沒有收到 id_dsa

1.ip位址

1. k0: 140.118.160.14
2. k1: 140.118.160.183
3. k2: 140.118.160.184
port=22

2.指令

1. 切換至superuser

k0:~ # su -
#su= super user
k0:~ # adduser
#加入新的cluster用戶
# 系統會將key檔(id_dsa)寄至學生信箱，再自行轉成私鑰檔(.ppk)

3.若mail沒有收到 id_dsa


#/home/{username}/.ssh/id_dsa
k0:~ # cp id_dsa /home/{your account}
k0:~ # chmod 777 id_dsa
#改變權限讓任何人都可以下載key檔
# 使用WinSCP下載key檔
