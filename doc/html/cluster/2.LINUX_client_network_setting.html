    1.進入設定畫面

    k0:~ # yast
    #開啟網路設定畫面
    選擇 Network Devices --> Network Seeting



    2.進入Edit(按[alt]+i)

    ip address: 192.168.0.1**
    #i01~i25為101~125；e01~e05為126~130
    Next



    3.Hostname

    Hostname=e** (or i**)
    name server 1= 192.168.0.1
    routing --> Default Gateway = 192.168.0.1
    # k0
    Quit







    4.重啟

    k0:~ # /etc/init.d/network restart

    5.測試是否成功設定

    k0:~ # ping www.google.com
    #對外部網站丟封包進行測試
    [ctrl]+C to cancle
