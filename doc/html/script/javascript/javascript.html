  • Reference

    • 1995年Netscape的Brendan Eich設計出JavaScript語言(原名為 LiveScript，為了與昇陽合作，搭上 Java 的順風車，改名為 JavaScript)，加入到 Netscape 2.0 瀏覽器中，因而成為瀏覽器的共通語言。JavaScript是一種原型導向的語言，具有動態與弱型別等特性，後來各家瀏覽器紛紛支援，但是由於實作方式和語法都不統一(JavaScript同時存在：Netscape Navigator 3.0的JavaScript、IE 中的JScript以及CEnvi中的ScriptEase)，因此造成了混亂的狀況，於是在 1998 年提交到 ECMA 組織制訂成 ECMA Script 的標準，以便統一 JavaScript 的語法。
    Web 2.0的風潮進一步刺激了JavaScript的廣泛使用，許多網站利用AJAX的JavaScript技術達成了高度的網頁互動性，像是Google Map 就是其中最著名的網站。在HTML5.0的草案提出之後，JavaScript更受到高度的重視，由於HTML5的強大功能，讓大家對JavaScript的發展充滿了信心，很多人認為JavaScript將會是繼C之後最重要的程式語言。

    變數型態

    JavaScript中的變數分為numerical、string、boolean和object type有array、object、resource、NULL
    JavaScript的變數宣告要使用var，變數名稱的開頭必須是底線（_）或英文字母，英文字母的大小寫都可以，而且大小寫有別。要注意數字不可以作為變數的開頭

    scalar

    // Numerical
    var num1 = 1; // 十進位
    var num2 = 0xff; // 十六進位
    var num3 = 033; // 八進位
    var num4 = 1234.5678; // 浮點數
    var num5 = 6.02e23; // 使用科學記號 6.02 x 10 23

    // 四捨五入
    num4.toFixed(0);
    num4.toExponential(1);
    num4.toPrecision(3);

    // Numerical轉String
    var str0 = num1 + ""; // 第一種方法
    var str0 = String(num1); // 第二種方法
    var str0 = num1.toString; // 第三種方法

    // String
    var str1 = 'ABC';
    var str2 = 'xyz';
    var str3 = '123';
    document.write(str1 + str2);

    // String轉Numerical
    Number(str3);
    parseInt(str3);
    parseFloat(str3);

    // String處理
    str1.toLowerCase();
    str2.toUpperCase();
    " lorem ".trim();
    " lorem".trimLeft();
    "lorem ".trimRight();
    "lorem ipsum".substr(6, 5);
    "value: " + 8;

    JSON JavaScript Object Notation
    scalar: string, numbers
    sequence: array, list
    mapping: key/value, hash, dictionary

    1） sequence數據之間用逗號（"，"）分隔。
    2） sequence數據集合（數組）用中括號("[]")表示。
    3） mapping數據用冒號（"："）表示用逗號（"，"）分隔。。
    4） mapping數據集合（對象）用大括號（"{}"）表示。

    Event Descritpion
    abort 圖像加載被中斷
    blur 元素失去焦點
    change 用戶改變域的內容
    click 滑鼠點擊某個對象
    dblclick 滑鼠雙擊某個對象
    error 當加載文檔或圖像時發生某個錯誤
    focus 元素獲得焦點
    keydown 某個鍵盤的鍵被按下
    keypress 某個鍵盤的鍵被按下或按住
    keyup 某個鍵盤的鍵被鬆開
    load 某個頁面或圖像被完成加載
    mousedown 某個滑鼠按鍵被按下
    mousemove 滑鼠移動
    mouseout 滑鼠從某元素移開
    mouseover 滑鼠被移到某元素之上
    mouseup 某個滑鼠按鍵被鬆開
    reset 重置按鈕被點擊
    resize 窗口或框架被調整尺寸
    select 文本被選定
    submit 提交按鈕被點擊
    unload 用戶退出頁面

    Level 0 DOM Event又稱Traditional registration model(傳統註冊模型), 事件處理以"on-"以開頭，語法為onevent。在event handler(事件處理器)後面直接實作Javascript，
    <body onload="var i = 1; i = i * 2; alert(i);">
    然而常用方式是直接呼叫函式
    <body onload="calcNum();">
    移除此函式則是在事件處理器呼叫null
    <body onload="null;">

    Level 2 DOM Event是使用在event listener(事件監聽器)去控制，語法如下：
    addEventListener('event', eventFunction, boolean);
    event:為事件如load, click等
    eventFunction:控制事件函式
    boolean:事件補獲機制，true為上到下的Capture phase(Netscape)，false為下到上的Bubbling phase(Internet Explorer)

    addEventListener 增加 removeEventListener 刪除 dispatchEvent 分派

    BOM

    在瀏覽器開始支援JavaScript時，存在一組物件提供有限的功能，在當時Netscape Navigator與Internet Explorer大戰的年代，兩個瀏覽器在這組物件上有些交集的部份，這些交集的部份被留存到現在，並在一些主流的瀏覽器中仍有支援。這組物件稱為BOM（Browser Object Model、瀏覽器物件模型）或非正式地稱為Level 0 DOM，因為它在DOM（Document Object Model）標準化前就已存在，而不是真有有文件規範Level 0 DOM。這組物件從全域物件window開始，具有以下的特性階層關係：
    window|
              |navigator
              |location
              |frames
              |screen
              |history
              |document
                           |forms
                           |links
                           |anchors
                           |images
                           |all
                           |cookie

    DOM

    W3C聯合各瀏覽器廠商制訂了標準物件模型，試圖讓各瀏覽器廠商遵合此一模式進行實作，以解決各瀏覽器間物件模型不一致的問題，在新的物件模型中，也對文件操作的功能加以擴充。簡單來說，在DOM的標準下，一份文件中所有的標籤定義，包括文字，都是一個物件，這些物件以文件定義的結構，形成了一個樹狀結構。例如：
    <html>
    <head>
      <title>首頁</title>
    </head>
    <body>
      <h1>Hello JavaScript!</h1>
      <a href="index.html">xxx</a>
    </body>
    </html>

    這份HTML文件，會形成以下樹狀的物件結構：
    document                                               (Document)
                |-html                                        (HTMLHtmlElement)
                        |-head                               (HTMLHeadElement)
                        |       |-title                         (HTMLTitleElement)
                        |             |-首頁                  (Text）
                        |
                        |-body                                (HTMLBodyElement)
                              |-h1                             (HTMLHeadingElement
                              |   |-Hello JavaScript       (Text)
                              |
                              |-a                               (HTMLAnchorElement)
                                |-xxx                          (Text)
