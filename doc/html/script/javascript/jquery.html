    basic selector

    Selector jQuery example Description
    Elements $('a') 選擇所有符合標籤名稱(Tag name)的元素
    ID $('#html') 選擇一個符合id屬性
    Class $('.web') 選擇所有符合class的元素

    Selector jQuery example Description

    descendant $('#web a') 底下所有層數之element

    all sibling $('#html ~ p') 之後所有element

    filter

    Filter jQuery example Description
    first, last $('p:first') 取出最初/最後的element
    even, odd $('p:even') 取出第奇數/偶數的element
    eq $('p:eq(2)') 取出第n個的element
    lt, gt $('p:gt(0)') 取出第n個之前/後的element
    contains $('p:contains("CSS")') 包含字串的element
    empty $('p:empty') 內容為空的element
    has $('p:has(a)') sub element
    first-child, last-child $('p:last-child') 取出最初/最後的child element
    nth-child $('#script > li:nth-child(4n)') 取出第n個的child element
    only-child $('p img:only-child') 取出唯一的child element
    header 取出所有head element
    input 取出所有form element(<input>,<textarea>,<button>,<select>)
    text 取出所有element
    password 取出所有element
    radio 取出所有element
    chcekbox 取出所有element
    file 取出所有element
    submit 取出所有element
    image 取出所有element
    reset 取出所有element
    hidden 取出所有element
    enabled, disabled 取出所有enabled/disabled狀態element
    hidden, visible 取出所有idden/visible狀態element
    chceked 取出所有checked狀態element
    selected 取出所有selected狀態element
    animated 取出所有animated狀態element
    not 取出不符合只element

    attributed seletor

    Condition jQuery example Description
    $('p[name]') 擁有特定attribute
    = $('p[name = "other"]') attribute等於特定條件
    = $('p[name != "other"]') attribute不等於特定條件
    ^= $('li[id ^= "p"]') attribute值開頭符合特定條件
    $= $('a[href $= ".org/"]') attribute值結尾符合特定條件
    *= $('a[href *= "Java"]') attribute值內容符合特定條件

    Function Description Example
    html html用為讀取和修改元素的HTML標籤 /* HTML */
    <p></p>

    /* jQuery */
    $('p').html('<b>Hello jQuery</b>');

    /* Result */
    <p><b>Hello jQuery</b></p>
    text 用來讀取或修改元素的純文本內容 /* HTML */
    <p></p>

    /* jQuery */
    $('p').text('<b>Hello jQuery</b>');

    /* Result */
    <p>&lt;b&gt;Hello jQuery&lt;/b&gt;</p>
    append 在匹配的元素內部最後面加入(內部插入) /* HTML */
    <p>jQuery</p>

    /* jQuery */
    $('p').append('<b>Hello</b>');

    /* Result */
    <p>jQuery<b>Hello</b></p>
    prepend 在匹配的元素內部最前面加入(內部插入) /* HTML */
    <p>jQuery</p>

    /* jQuery */
    $('p').prepend('<b>Hello</b>');

    /* Result */
    <p><b>Hello</b>jQuery</p>
    after 在匹配的元素外部後面加入(外部插入) /* HTML */
    <p>jQuery</p>

    /* jQuery */
    $('p').after('<b>Hello</b>');
