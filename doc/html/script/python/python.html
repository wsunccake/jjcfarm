    Python是物件導向(object-oriented)程式級高階程式語言，也是直譯式(shell script)程式語言。Python強調對程式語言的語句易讀、易懂、易學(簡潔和清晰的語法特點)及加快程式開發的時效，方便使用，可以完成各種難度的應用，並可在大多數的系統中運行，以減少開發及維護成本的觀念進行發展。Python 同時支援 modules 和 packages ，另外 Python 為跨平台程式語言也支援 unicode 字元。功能強大而完善的通用型語言，可以用於很多種軟體開發動態程式，使得 Python 非常有吸引力，發展至今已有十多年的歷史，成熟且穩定。Python具有腳本語言中最豐富和強大的類庫，足以支持大多數日常應用。它的名字來源於一個喜劇，也許最初設計Python這種語言的人並沒想到今天Python會在工業和科研上獲得如此廣泛的使用。著名的自由軟件作者Eric Raymond在他的文章《如何成為一名黑客》中，將Python列為黑客應當學習的四種程式語言之一，並建議人們從Python開始學習程式。這的確是一個中肯的建議，對於那些從來?有學習過程式或者並非計算機專業的程式學習者而言，Python是最好的選擇之一。
    創始人為Guido van Rossum。在1989年聖誕節期間的阿姆斯特丹，Guido為了打發聖誕節的無趣，決心開發一個新的腳本解釋程序，作為ABC語言的一種繼承。之所以選中Python大蟒蛇的意思）作為程序的名字，是因為他是一個Monty Python的飛行馬戲團的愛好者。ABC是由Guido參加設計的一種教學語言。Guido本人看來，ABC 這種語言非常優美和強大，是專門為非專業程序員設計的。但是ABC語言並沒有成功，究其原因，Guido認為是非開放造成的。Guido決心在Python中避免這一錯誤（的確如此，Python 與其它的語言如C、C++和Java結合的非常好）。同時，他還想實現在 ABC 中閃現過但未曾實現的東西。就這樣，Python在Guido手中誕生了。實際上，第一個實現是在Mac機上。可以說Python是從ABC發展起來，主要受到了Modula-3（另一種相當優美且強大的語言，為小型團體所設計的）的影響。並且結合了Unix shell和C的習慣。早期Python是以C語言為基底撰寫而成，目前則有以Java語言為基底的Jython及架構在.NET的IronPython版本，甚至有以Python語言為基底的PyPy實作品，可將 Python程式碼編譯成其他語言的執行檔，像是C、CLI(.Net)、JavaScript等。Guido van Rossum在2005年底成為Google公司雇員，該公司將Python列為程式員的三大官方語言之一，意指允許員工使用 Python 實作產品與服務。

    #!/usr/bin/env python

    class UserInfo: #建立UserInfo類別
        def __init__(self, F = "/etc/passwd"): #constructure/建構式/初始化
            self.UID = []
            self.GID = {}
            self.User = {}
            self.Home = {}
            self.Shell = {}
            self.file(F)

        def __len__(self):
            return len(self.UID)

        def add(self, uid, gid, user, home, shell):
            self.UID.append(uid)
            self.GID[uid] = gid
            self.User[uid] = user
            self.Home[uid] = home
            self.Shell[uid] = shell

        def delete(self, uid):
            self.Shell.pop(uid)
            self.Home.pop(uid)
            self.User.pop(uid)
            self.GID.pop(uid)
            self.UID.remove(uid)

        def file(self, File = "/etc/passwd"):
            f = open(File)

            for l in f.readlines():
                uid = l.split(':')[2]
                gid = l.split(':')[3]
                user = l.split(':')[0]
                home = l.split(':')[5]
                shell = l.split(':')[6].rstrip()
                self.add(uid, gid, user, home, shell)

            f.close()

        def __del__(self): #deconstructure/解構式/結束化
            pass

    if __name__ == "__main__":
        userInfo = UserInfo() #產生一個UserInfo類別的物件

        User = []
        SystemUser = []
        NoLogin = []

        for u in range(len(userInfo)):
            id = userInfo.UID[u]

            if int(id) > 999:
                User.append(userInfo.User[id])
            else:
                SystemUser.append(userInfo.User[id])

            if userInfo.Shell[id] == '/bin/false':
                NoLogin.append(userInfo.User[id])

        print "User : " + ", ".join(User)
        print "System User : " + ", ".join(SystemUser)
        print "No Login User: " + ", ".join(NoLogin)

    inheritance/繼承

    python的類別沒有特別去分public和private，一般class宣告的method都為public，而名稱已__開頭的宣告method才為private。在python的繼承中，public method會被繼承，但是private method並不會。無論public或是private attribute，在使用上都可以任意指定（因為python是個動態宣告的程式語言），但這是不好的程式設計習慣。

    #!/usr/bin/env python

    class Animal():
        def __init__(self, name = None):
            self.__NAME__ = name
            print "Construct Animal"

        def setAge(self, age):
            self.__AGE__ = age

        def setName(self, name):
            self.__NAME__ = name

        def getName(self):
            return self.__NAME__

        def getAge(self):
            return self.__AGE__

        def __getHello__(self):
            print "Hello"

    class Felidae(Animal): #Felidae繼承Animal
        def __init__(self):
            Animal.__init__(self)
            print "Construct Felidae, and inheritance Animal"

    class Cat(Felidae): #Cat繼承Felidae
        def __init__(self):
            print "Construct Cat"

        getHello(self):
            super(Felidae, self).__getHello__() #取得Felidea父類別的method

    if __name__ == '__main__':
        c = Cat()
        c.__NAME__ = 'kitten' #python並不支援private attribute，所以就算不透過method也可存取
        print c.__NAME__
        print c.getName()
        c.setAge(1)
        print c.getAge()
        c.__ABC__ = '123' #動態產生變數
        print c.__ABC__
        c.__getHello__()

        f = Felidae()
