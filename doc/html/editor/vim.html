    基本模式

    常態模式（Normal mode）

    進入vim就是處於常態模式，只能下按鍵指令，不能輸入編輯文字。這些指令可能是游標移動的指令，也可能是編輯指令或尋找替換指令。


    插入模式（Insert mode）

    按i就會進入插入模式，此時才可鍵入文字，編輯文章，按Esc會回到常態模式。在狀態列會有 - INSERT - 字樣。


    命令列模式（Cmdline mode/Command-line mode）

    按冒號：會進入命令列模式，左下角會有一個冒號：出現可下指令。按Esc回命令列模式。反正正常狀態都是處於常態模式，這樣才不會把您辛苦打字的文章給隨便搞亂掉。搜尋時的 / 及 ? 按鍵也是屬於命令列模式。


    反白模式（Visual mode）

    這就是把需要處理的文字反白起來，等待處理的模式，底部狀態最會有 - VISUAL - (從Normal mode輸入v進入)或 - VISUAL LINE - (從Normal mode輸入V進入) 或 - VISUAL BLOCK - (從Normal mode輸入^v進入)字樣。


    選擇模式（Select mode）

    類似Windows操作模式，可直接進入Insert mode。底部狀態最會有 - SELECT - (從Normal mode輸入gh進入)或 - VISUAL LINE - (從Normal mode輸入gH進入) 或 - VISUAL BLOCK - (從Normal mode輸入g^h進入)字樣。


    Ex模式（Ex mode）

    和命令列模式是一樣的，但下了命令後不會返回，會留在行ex編輯器狀態，有一個冒號在那兒，等待輸入命令。由 gQ 按鍵可以進入 Ex 模式，輸入 visual 可以返回正常模式。


    額外模式（additional mode）

    除了上述幾個常見的模式，還有以下幾個少用到的模式，這裡稱為額外模式


    取代模式（replace mode）

    下R指令時所處的狀態。狀態列會有 - REPLACE - 字樣。


    插入常態模式（insert normal mode）

    特殊模式，在插入模式時，進入輸入狀態，但按 Ctrl+O 就會進入插入常態模式，和常態模式一樣，只不過執行完所下的指令後又會馬上返回原來的插入模式繼續輸入文字。狀態列會有 - (insert) - 字樣，是小寫有小號的。


    插入反白模式（insert visual mode）

    這和插入常態模式一樣，只不過在按 Ctrl+O 後所執行的是反白的 Ctrl+V 或 V 或 v 而進入反白模式，等反白模式結束又會返回原來的插入模式。狀態列會有 - (insert) VISUAL - 字樣。


    插入選擇模式（insert select mode）

    這和插入反白模式一樣，只不過進入的是選擇模式，而非反白模式。狀態列會有 - (insert) SELECT - 字樣。


    基本操作

    基本移動操作

    normal mode description
    h、→、Backspace 向左移動一個字元
    j、↓、Enter、+ 向下移動一個字元
    k、↑、 - 向上移動一個字元
    l、→、Space 向右移動一個字元
    [Ctrl]^f、PageDown 向下/前翻頁（Forward）
    [Ctrl]^e 向下/前翻頁，但游標固定行數
    [Ctrl]^d 向下/前翻頁，但游標固定位置
    [Crtl]^b、PageUp 向上/後翻頁（Backwad）
    [Ctrl]^y 向上/後翻頁，但游標固定行數
    [Ctrl]^u 向上/後翻頁，但游標固定位置
    w 移至次一個字（word）字首
    W 同上，但會忽略一些標點符號
    e 移至後一個字字尾
    E 同上，但會忽略一些標點符號
    b 移至前一個字字首
    B 同上，但會忽略一些標點符號
    ) 移至下一個句子（sentence）首
    ( 移至上一個句子（sentence）首
    { 移至下一個段落（paragraph）首
    } 移至上一個段落（paragraph）首
    % 匹配 {}，[]，()
    0、Home 移至行首（含空白字元）
    ^ 移至行首第一個非空白字元
    $、End 移至行尾
    G 移至檔尾（全文最後一行的第一個非空白字元處）
    gg 移至檔首（全文第一行之第一個非空白字元處）
    n| 移至第n個字元
    nG 移至第n行行首
    H 移至螢幕頂第一個非空白字元
    M 移至螢幕中間第一個非空白字元
    L 移至螢幕底第一個非空白字元
    command mode description
    :e[dit] 開啓檔案、:e text.txt(開啓text.txt)、:e!(重開檔案，用於修改後但未儲存)
    :w[rite] 儲存檔案、:w(存檔)、:w newtext.txt(另存新檔)
    :q[uit] 離開檔案、:q(離開)
    :r[ead] 讀入檔案、:r tmp.txt(插入tmp.txt內容到檔案)、:r !ls(從stdin插入到檔案)

    編輯指令

    基本編輯

    normal mode description
    i 在游標所在字元前開始輸入文字(insert)
    a 在游標所在字元後開始輸入文字(append)
    o 在游標所在行下開一新行來輸入文字(open)
    I、^i、0i 在行首開始輸入文字
    A、$a 在行尾開始輸入文字
    O 在游標所在行上開一新行來輸入文字
    J 將下一行整行接至本行(Joint)
    x 刪除游標所在處之字元
    X 刪除游標前之字元
    dd 刪除一整行(delete line)
    dw 刪除一個字(delete word)
    dgg 刪至檔首
    dG 刪至檔尾
    dnG 刪至檔中第n行
    D、d$ 刪至行尾
    d0、d^ 刪至行首
    r 取代游標所在處之單一字元
    R 進入取代模式（replace mode）
    S、cc 取代整行內容
    cw 替換一個英文字(word)
    ~ 游標所在處字元之大小寫互換
    C、c$ 取代至行尾，即游標所在處以後的字都會被替換
    c0、c^ 取代至行首
    s 替換一個字元為您所輸入的字串
    u 回復所有編輯動作(undo)
    U 在游標沒離開本行之前，回復所有編輯動作
    Crtl+r redo
    ndd 刪除游標所在處（含）起算以下n行
    nr 取代n個字元
    nJ 將n行合併成一行
    nx 刪除n個字元
    yy、Y 複製整行至緩衝（buffer）
    nyy、yny 複製游標所在處（含）起算以下n行至緩衝
    y^ 複製游標所在處行到首（不含游標）至緩衝
    y0 複製游標所在處行到首（不含游標）至緩衝
    y$ 複製游標所在處行到尾（含游標）至緩衝
    ygg 複製游標所在處到檔案首至緩衝
    yG 複製游標所在處到檔案尾至緩衝
    ynw 複製n個word至緩衝
    ynW 複製n個word至緩衝
    p 貼上緩衝游標後
    P 貼上緩衝游標前
    "cY 複製至名稱為c的緩衝
    "cy 複製至名稱為c的緩衝
    "cp 貼上名稱為c的緩衝游標後
    "cP 貼上名稱為c的緩衝游標前
    gUw、gU$ 小寫轉換大寫(w是單字, $是整行)
    guw、gu$ 大寫轉換小寫(w是單字, $是整行)

    screen split（螢幕分割）

    command mode visual mode descritpion
    :sp[lit] [Ctrl]^w + s 水平分割
    :vsp[lit] [Ctrl]^w + v 垂直分割
    [Ctrl]^w + h, j, k, l 切換到左, 下, 上, 右螢幕
    :res[ize] [Ctrl]^w + +, - 增加/減少水平分割螢幕尺寸
    :vertical resize [Ctrl]^w + <, > 增加/減少垂直分割螢幕尺寸
    [Ctrl]^w + _ 目前分割螢幕最大化
    [Ctrl]^w + = 平均分配分割螢幕
    [Ctrl]^o
    [Ctrl]^^ 切換檔案（目前已有多個檔案被開啟）
    q: 命令紀錄

    vim -Om指令時，在啓動時會先垂直分隔m個screen。
    vim-on指令時，，在啓動時會先水平分隔n個screen。
    在多個檔案被開啟，可使用:ls、:ls!或:bu[ffer]顯是目前有開啟的檔案，使用:bu[ffer] n直接切換到該檔案。


    folding（折疊）

    normal mode command mode descritpion
    :.,+n fol[d] zfnj 從本行算起到第n行折疊
    zd 刪除折疊
    :foldo[pen] zo 開啟折疊（只有目前選擇的折疊）
    :foldc[lose] zc 關閉折疊（只有目前選擇的折疊）
    za 開/關折疊（只有目前選擇的折疊）
    :foldo[pen]! zO 開啟折疊（巢式折疊以遞迴處理）
    :foldc[lose]! zC 關閉折疊（巢式折疊以遞迴處理）
    zA 開/關折疊（巢式折疊以遞迴處理）
    zj 搜尋上一個折疊
    zk 搜尋下一個折疊
    zm 關閉下一層折疊（文件中所有折疊）
    zr 開啟下一層折疊（文件中所有折疊）
    zM 關閉所有折疊（文件中所有折疊）
    zR 開啟所有折疊（文件中所有折疊）

    foldcolumn設定顯示折疊欄值為0~12，預設為0（不顯示）;foldmethod設定折疊方式，值可為，預設為mamual
    manual:以標準vim結構定義摺疊跨越的範圍，類似移動命令
    indent:摺疊與摺疊層級，對應制文字縮排與shiftwidth選項值
    expr:以正規表示定義摺疊
    syntax:摺疊對應制檔案使所用的程式語言
    diff:摺疊兩個檔案間的差異
    maker:預先在檔案中定義標記（可由使用者定義），指定摺疊邊界

    variables

    symbol descritpion
    " comment，註解（僅限使用在句首）
    | 斷行（在vim中;另有功能）
    \ 接續（僅限使用在句首）
    non-prefix variable The variable is global、變數範圍為全域
    g: The variable is global、變數範圍為全域
    s: The variable is local to the current script file、變數範圍為當前的腳本文件
    w: The variable is local to the current editor window、變數範圍為當前的編輯器窗口
    b: The variable is local to the current editor buffer、變數範圍為當前的緩衝
    t: The variable is local to the current editor tab、變數範圍為當前的編輯器選項卡
    l: The variable is local to the current function、變數範圍為當前的函數
    a: The variable is a parameter of the current function、變數範圍為函數參數
    v: The variable is one that Vim predefines、Vim的預定義變數
