    Sublime Text 是款強大的跨平台文字編輯器, 可支援 Windows、OS X、Linux 多個系統, 他的開發者是前 Google 軟體工程師 Jon Skinner, 據說受到 Vim 的種種啟發, 決定打造一個更好的文字編輯器. 他秉持著三項基本原則來開發 Sublime Text:

    • 專注在文字與程式碼上, 而不是讓人眼花撩亂的工具列
    • 對話框訊息不使用晦澀難懂的文字
    • 善用螢幕的每個空間, 使全螢幕、多螢幕模式一起編輯檔案, 儘可能很容易

    Sublime Text 可以減少撰寫程式碼許多重複性的工作，讓開發者們可以更專注在程式身上，而且過程是讓人感到輕鬆、愉悅、超爽的！

    Windows Linux Mac
    column mode [Ctrl] ^ [Alt] + ↑、↓ [Shift] ^ [Alt] + ↑、↓ [shift] ^ [control] + ↑、↓
    command palette [Ctrl] ^ [Shift] + p [shift] ^ [command] + p
    show/hide side [Ctrl] + k, [Ctrl] + b [command] + k, [command] + b
    show console [Ctrl] + ` [control] + `
    goto anything/file [Ctrl] + p [command] + p
    goto symbol/function [Ctrl] + r [command] + r
    goto line [Ctrl] + g [control] + g
    goto next find [Ctrl] + i [command] + i
    goto prev find [Ctrl] ^ [Shift] + i [command] ^ [shift] + i
    goto next file [Ctrl] + PageDown [command] ^ [shift] + ]
    goto prev file [Ctrl] + PageUp [command] ^ [shift] + [
    goto next in stack [Ctrl] + [Tab] [control] + [tab]
    goto prev in stack [Ctrl] ^ [Shift] + [Tab] [control] ^ [shift] +[tab]
    folding close [Ctrl] ^ [Shift] + [ [command] ^ [option] + [
    folding open [Ctrl] ^ [Shift] + ] [command] ^ [option] + ]
    folding level n (n = 1 ~ 9) [Ctrl] + k, [Ctrl] + n [command] + k, [commans] + n
    selection line [Ctrl] + l [commands] + l
    selection word [Ctrl] + d [command] + d
    selection bracket [Ctrl] ^ [Shift] + m [commads] ^ [shift] + m
    selection indentation [Ctrl] ^ [Shift] + j [commads] ^ [shift] + j
    selection tag [Ctrl] ^ [Shift] + a [commads] ^ [shift] + a

    Packag Control

    Package Control 由 Will Bond 所開發, 是功能相當完整的 package manager, 讓搜尋, 安裝跟移除 Sublime Text package 變得相當方便, 協助自動更新外掛, 也因此幾乎所有的外掛開發者, 都會將自己的作品提交到 Package Control 上. 因為是太方便了, 所以幾乎與 Sublime Text 緊密結合在一起, 成為安裝 Sublime Text 時第一個必裝的 package.


    ref: Pacakge Control


    Emmet

    Emmet前身為Zen Coding, 其作用是利用已有的技術, 高速的架構出HTML, CSS, XML, XSL或HAML. 什麼是已有的技術, 簡單說, 它是CSS Selector (CSS選擇器) 的再利用. 就算只是簡單拿來產出HTML5或CSS3的程式碼, 也可非常有效率的加速寫HTML5與CSS3的速度.


    ref Emmet


    Emmet LiveStyle

    LiveStyle 可幫 developer 邊 coding 邊在 browser 預覽效果, 但不只這樣而已, 最大的特點包含:

    • 完全不用存檔 (在 web developer 寫 code，他也會幫你寫回檔案)
    • 不用重新整理葉面
    • 完全做到雙向互通
    • 跨平台
    • 創新的 CSS patchin
    • 即時更新多視覺 & 多裝置



    ref Emmet LiveStyle
