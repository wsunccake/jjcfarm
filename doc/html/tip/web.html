    CGI

    introduction

    CGI的完整名字是Common Gateway Interface. 它是一種標準介面程式, 能讓你的網頁跟 WWW server 溝通, 達到跟使用者互動的效果. 透過CGI, 可產生動態網頁, 秀出 server 上的最新資料. 當link到一個CGI物件時, 取回的文件並非是一份靜態資料, 而是藉由程式動態產生的HTML資料流. 傳回來的資料也許分分秒秒都在更改, 或者針對使用者的特殊查詢而有不同的反應. 例如股票市場行情等等.
    CGI並非是什麼特定語言, 事實上只要是在 WWW Server 上的執行檔, 皆可當作CGI 程式. 所以用什麼語言來撰寫 CGI 程式都可.

    Linux:~ # cat /cgi-bin/hello.sh
    #!/bin/sh

    cat << EOF
    Content-type:text/html

    <html>
    <body>
    <p>Run SH CGI</p>
    </body>
    </html>
    EOF


    Linux:~ # cat /cgi-bin/hello.pl
    #!/usr/bin/env perl

    print <<EOF
    Content-type:text/html

    <html>
    <body>
    <p>Run Perl CGI</p>
    </body>
    </html>
    EOF


    Linux:~ # python /cgi-bin/hello.py
    #!/usr/bin/env python

    print '''Content-type:text/html

    <html>
    <body>
    <p>Run Python CGI</p>
    </body>
    </html>'''

    HTTP request method: get & post

    CGI程式與一般的程式最大的差別在於資料擷取的不同, 網頁資料的擷取上主要有兩種, 分別是透過GET與POST. 這兩種可以透過表單函數<form method="GET/POST">設定.
    GET: 在網址列後面輸入變數(例如 ?var1=val1&var2=val2)即可觀察到變數 Query_String.
    POST: 在CGI程式之中，會被當做是Standard_input也就是當成外在輸入.

    Linux:~ # cat /cgi-bin/get.sh # curl http://127.0.0.1/cgi-bin/get.sh?name_sh=john&age_sh=10
    #!/bin/sh

    if [ x$REQUEST_METHOD == xGET ]
    then
      for data in `echo $QUERY_STRING | tr "&" "\n"`; do
        key=`echo $data | awk -F= '{print $1}'`
        val=`echo $data | awk -F= '{print $2}'`
        if [ $key == name_sh ]
        then
          name_sh=$val
        fi
        if [ $key == age_sh ]
        then
          age_sh=$val
        fi
      done
    fi

    echo "Content-type:text/html"
    echo ""
    echo "Hello ${name_sh}."
    echo "<br />You are ${age_sh} years old."


    Linux:~ # car /cgi-bin/get.pl
    #!/usr/bin/env perl

    my ($data, @datas, @keys, %FORM);
    if ($ENV{'REQUEST_METHOD'} == 'GET') { $data = $ENV{'QUERY_STRING'}; }

    @datas = split(/&/, $data);
    $i = 0;
    foreach $d (@datas) {
      @tmp = split(/=/, $d);
      $keys[$i] = $tmp[0];
      $FORM{$tmp[0]} = $tmp[1];
      $i++;
    }

    print "Content-type:text/html\n\n";
    print "Hello " . $FORM{'name_pl'} . ".\n";
    print "<br />You are " . $FORM{'age_pl'} . " years old.";


    Linux:~ # cat /cgi-bin/get.py
    #!/usr/bin/env python

    import os
    keyL = []
    FORMH = {}
    if os.getenv('REQUEST_METHOD') == 'GET':

            keyL.append(tmp[0])

    print "<br />You are ", FORMH['age_py'], " years old.\n"


    Linux:~ # cat /cgi-bin/post.sh # curl -X POST http://127.0.0.1/cgi-bin/post.sh --data "NAME_SH=john&AGE_SH=18"
    #!/bin/sh

    if [ x$REQUEST_METHOD == xPOST ]; then
      read datas
      for data in `echo $datas | tr "&" "\n"`
      do
        key=`echo $data | awk -F= '{print $1}'`
        val=`echo $data | awk -F= '{print $2}'`
        if [ $key == NAME_SH ]
        then
          NAME_SH=$val
        fi
        if [ $key == AGE_SH ]
        then
          AGE_SH=$val
        fi
      done
    fi

    echo "Content-type:text/html"
    echo ""
    echo "Hello ${NAME_SH}."
    echo "<br />You are ${AGE_SH} years old."

    }

    print "<br />You are " . $FORM{'AGE_PL'} . " years old.";

    CSS selector

    selector CSS HTML Description
    class .emText { color: red;} <p class="emText">此段會套用紅色</p> 以.開頭, 名稱可自訂, 一個網頁可有多個class屬性值
    id #emText { color: red;} <p id="emText">此段會套用紅色</p> 以#開頭, 名稱可自訂, 一個網頁只能有一個ID屬性值
    element h1 {color: red;}
    p {color: black;}     		<h1>標題會套用紅色</h1>
    <p>段落會套用黑色</p>     		設定於HTML的標籤上, 網頁上所有的標籤都會套用
    universal * {color:red;} <h1>標題會套用紅色</h1>
    <p>段落會套用黑色</p>    		 使用字元*, 整個網頁下的所有元素都會套用設定
    attribute p[name] {color:red;} 擁有特定attribute做CSS設定
    p[name = "other"] {color: green;} attribute等於特定條件
    'p[name != "other"] attribute不等於特定條件
    li[id ^= "per"] attribute值開頭符合特定條件
    a[href $= ".org/"] attribute值結尾符合特定條件
    a[href *= "Java"] {color: yellow;} attribute值內容符合特定條件
    descendant E F, 利用空白鍵區隔兩個元素, 表示在 E 元素內的 F 元素才會套用
    child
    adjacent sibling
    general sibling E ~ F, 利用~區隔兩個元素，表示在與 E 同一層關係的 F 元素全部都會套用
    groups 以,區隔, 同時對多個選擇器定義樣式
    E, F, G，同時針對E、F、G元素設定樣式
