    使用環境

    windows環境:C:\Pythonxy;C:\Pythonxy\Lib\site-packages;C:\Pythonxy\Scripts
    Unix-Like環境:/usr/lib/pythonx.y/site-packages

    command/指令

    slackware:~ # python -c "print 'Hello, Python'"
    Hello, Python

    environmental variable/環境變數

    引號

    variable description
    PYTHONPATH 設定module path，預設為/usr/lib/python<version>/site-packages
    PYTHONHOME 設定模組搜尋路徑相關的變數，預設為/usr/lib/python<version>
    PYTHONSTARTUP 設定進入interactive mode前所執行程式路徑

    slackware:~ # python -c 'import sys, pprint; pprint(sys.path)' # 顯示目前PYTHONPATH設定
